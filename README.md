# Meal Generator App - created with ReactJs

for a live Demo: [Live Demo]('')

## Features

- React-Router
- API calls
  - axios
- Get a randomized meal including image, recipe, ingridients and video guide
- Search functionality to search for a meal
- Chakra-UI
- Light/Dark Mode
- Mobile support
